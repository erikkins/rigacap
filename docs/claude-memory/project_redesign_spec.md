**Source:** `/Users/erikkins/Downloads/rigacap-redesign-reprice-session-notes.md` — read this for full context.

**Core reframe:** RigaCap is an editorial financial publication that happens to deliver signals, not a signal service. Value prop = external discipline, not signals.

**Pricing:** $129/mo, $1,099/yr, $59/mo founding (first 100, locked 12mo)

**Design system:**
- Paper: `#F5F1E8`, Card: `#FAF7F0`, Ink: `#141210`, Accent (Claret): `#7A2430`
- Typography: Fraunces (display) + IBM Plex Sans (body) + IBM Plex Mono (data)
- Aesthetic: Grant's / Economist / FT — NOT trading app
- No gradients, no bright greens, no pill buttons, no "BUY" in caps
- "Record Entry" replaces BUY buttons

**Hero copy:** "A disciplined momentum strategy that stays in cash when it should — and exits before major drawdowns. Built for the investor who's tired of fighting their own worst instincts."

**Must-fix before launch:** securities attorney consult, fix arithmetic inconsistencies, intraday execution disclosure, balanced social content rule

**UI priority order:** (1) kill Journey gradient, paper background, typography swap, soften buttons (2) regime bar, stat cards, chart palette (3) header, admin, trade history
