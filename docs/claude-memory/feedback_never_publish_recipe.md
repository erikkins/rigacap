**The rule:** Public surfaces (website, social cards, posts, emails to non-NDA'd audiences) describe the signal's STRUCTURE ("price breaking out above its long-term weighted average", "momentum accelerating inside an established trend, volatility-penalized", "strength near recent highs") but NEVER the coefficients (DWAP ×1.05, 5-day/60-day weights 0.3/0.2, volatility penalty 0.15, within-3%-of-50-day-high, $15/500k floors as signal params). Exception: risk-management parameters already public by decision (30% trailing stop, ~20 positions, turnover stats). Private diligence docs in design/documents/ may carry the full formula.

**Why:** The methodology page was deliberately "de-recipe'd" June 9, 2026. On June 11 a launch-card canon brief leaked the exact thresholds onto public social cards; Erik: "We said... NEVER SHOW THE FORMULAS." The recipe is the IP; publishing it invites free replication and undercuts the subscription.

**How to apply:** Any time canon numbers are propagated to a new surface, classify the surface public/private FIRST. Briefs to subagents must state which class the target is and explicitly forbid coefficients on public ones.
