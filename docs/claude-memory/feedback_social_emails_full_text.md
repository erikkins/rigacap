Social post approval emails must show the FULL post text, not a truncated preview.

**Why:** Erik thought there was an error in a post because the email truncated it at 300 chars with an ellipsis. He needs to see the complete content to make a cancel/approve decision.

**How to apply:** If adding new notification emails for social posts, always include the full text_content. Never truncate post previews in admin-facing emails.
